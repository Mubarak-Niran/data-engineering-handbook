# Interview Preparation & Resume Tips

## Common Data Engineering Interview Questions
### SQL Questions
- Write a query to find duplicate records in a table.
- Optimize a query that joins two large tables.

### Python & Data Processing
- How would you handle missing data in a dataset?
- Explain multiprocessing vs. multithreading in Python.

### System Design
- Design a real-time analytics system for a streaming application.
- Explain how you would optimize a data pipeline for cost efficiency.

## Resume Tips
- Focus on **projects that showcase hands-on data engineering work**.
- Highlight **SQL, Python, and Cloud experience**.
- Showcase **ETL pipelines, automation, and infrastructure projects**.
